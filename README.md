# Contributte

:squirrel: Contributte Website & Docs

## Build

```
# Install dependencies
npm install
```

*Just once*

```
# Clone repositories
node tools/git-tools.js

# Generate pages from readmes
node tools/generate-pages.js 
```

```
# Start dev server 
npm run dev
```

**Production**

```
# Compile source
npm run build

# Start prod server
npm start
```

**Generate static**

```
# SSR
npm run generate
```
