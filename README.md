# Contributte

:squirrel: Contributte Website & Docs

## Dev

```
# Install dependencies
npm install
```

*Just once*

```
# Clone repositories
node tools/git-tools.js

# Generate pages from readme(s)
node tools/generate-pages.js 
```

```
# Start dev server 
npm run docs:dev
```

**Production**

```
# Compile sources
npm start docs:build
```
